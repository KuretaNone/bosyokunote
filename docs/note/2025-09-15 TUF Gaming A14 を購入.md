# 2025-09-15 TUF Gaming A14 を購入

祖父が使用しているPCが古く、Windows11に非対応でした。
そのため、現在自分が使用してるデスクトップを祖父に譲渡し、自分はPCを買い換えようということでPCの買い換えを検討しました。

家の空間を結構でかでかと占領してるデスクトップはもうこりごりだなぁと思ったのでノートPCに移行します。

## 求めるスペック

- 床にデスクトップを置くよりも掃除がしやすい
- 前回よりもいいスペックのPC
    - Core i7 11世代
    - RTX 3060
    - メモリ16GB
- 持ち運びを考慮する上で、本体が大きすぎないこと
    - 最終的には14インチで探してた
- 重量がおおむね1.5kgに収まること
- HHKBユーザなのでテンキーレスのキーボードであること
    - のちに打ち箸を購入したので、これは無意味な考慮だったかなとも思う

## 探す上での問題点

スペックがいいPCとテンキーレスを実現しようとすると実は候補は一気に少なくなります。

> [!NOTE]
> スペックがいいPCをかおうとすると必然的に本体が大きくなるので、テンキーレスとなると選択肢が狭くなるわけです。

そして前持っていたPCよりもスペックがよいものを買う、ということは、少なくとも**RTX3060以上の性能を持ったPC**がほしいということです。
予算を大体20万円くらいを想定いているためその前後でPCを買おうとすると今回した**TUF Gaming A14**に落ちついたわけです。

## TUF Gaming A14 について

- 本体重量は1.5kg
- CPU AMD Ryzen AI 7 350
- メモリ 32GBGB
- GPU RTX 5060 Laptop 
- 画面 １４インチ

もう理想ですね。

M.2 SSD の容量が一番上のグレードじゃないと1TBないのがちょっと難点でしたが、
最終的には 25万円で本体を購入できたので非常に良いかいものでした。
