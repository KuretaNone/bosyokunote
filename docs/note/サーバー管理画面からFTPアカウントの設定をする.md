# サーバー管理画面からFTPアカウントの設定をする

## サーバー側の設定

[[XServer]]の管理画面からFTPアカウント設定を選択

FTPアカウントを追加を選択

![1748679729.png](res/1748679729.png)

- ユーザーID
- パスワード
- 接続先ディレクトリ
- FTP容量を入力(0だと無制限)
