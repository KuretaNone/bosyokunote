# DerivedDataの出力先を変更する方法

1. [[Xcode]]を起動する
2. `⌘ + ,`を押す
3. `Locations`を選択する
4. `Derived Data`の右側にある`Advanced`から設定を変更する。

> [!NOTE]
> 現在の設定はUnique。プロジェクトフォルダの直下に`DerivedData`フォルダが作成される。
