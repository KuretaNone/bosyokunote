# シンボリックリンク

シンボリックリンク（symbolic link）は、**実際のファイルやディレクトリへの参照**ができる

- リンクを開くと、実際にはターゲットのファイルやディレクトリを使用する
- ターゲットが削除されるとリンクは**「壊れたリンク」**（無効）になる
