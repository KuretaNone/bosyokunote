# 呼び出し規約(c)

関数の引数の受け渡し方法を指定するための規約。
[[Visual Studio]]廃止された規約もあるが、[[C++Builder]]では、使用できる規約もある。

例：
- `BOOL`
- `FAR`
- `PASCAL`
- `WINAPI`
