# Windowsでシンボリックリンクを作成する

[[Windows]]で [[シンボリックリンク]]を作成するには以下のようにする

```
mklink /d <link> <target>
```

- https://learn.microsoft.com/ja-jp/windows-server/administration/windows-commands/mklink
