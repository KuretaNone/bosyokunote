# E138 All main.shada.tmp.X files exist, cannot write ShaDa file

[[Neovim]]を終了するときに発生するエラー。

## 原因

キャッシュファイルが壊れてそう。

## 対策

[[Shadaφファイルの格納ディレクトリ]]を削除する。
