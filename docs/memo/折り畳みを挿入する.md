# 折り畳みを挿入する

折り畳みを挿入するには、`<details>`タグを使用します。

```markdown
<details>
<summary>これは中身が整形されない</summary>
1. hoge
2. fuga
3. piyo
    - foo
</details>

<details>
<summary>これは中身が整形される</summary>

1. hoge
2. fuga
3. piyo
    - foo
</details>
```

## 結果 

<details>
<summary>これは中身が整形されない</summary>
1. hoge
2. fuga
3. piyo
    - foo
</details>

<details>
<summary>これは中身が整形される</summary>

1. hoge
2. fuga
3. piyo
    - foo
</details>
