# WinMergeのショートカット

よく使うのだけ。

| 操作              | 結果             |
| --------------    | ---------------  |
| ドラック&ドロップ | 比較対象の追加   |
| F7                | 一個前の差分     |
| F8                | 一個後の差分     |
| F9                | マージモード切替 |
